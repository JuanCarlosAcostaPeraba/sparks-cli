# Commands

## List

```bash
sparks
sparks list
sparks list --all
sparks list --json
```

## Add

```bash
sparks add "Create GoReleaser config"
sparks + "Create Homebrew tap"
```

## Complete

```bash
sparks done 3
sparks ok 3
```

## Important

```bash
sparks important 3
sparks ! 3
```

## Remove

```bash
sparks remove 3
sparks rm 3
sparks - 3
```

## Clear

```bash
sparks clear
sparks clear --all --yes
```

## Tree

```bash
sparks tree
sparks tree --json
```

## Search

```bash
sparks search "codex"
sparks search "codex" --json
```

## Version

```bash
sparks version
```
