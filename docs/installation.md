# Installation

## Homebrew

```bash
brew tap JuanCarlosAcostaPeraba/tap
brew install sparks
```

Or:

```bash
brew install JuanCarlosAcostaPeraba/tap/sparks
```

## Manual

Download a release archive for your platform, extract it, and place `sparks` on your `PATH`.

## Windows

Download the Windows archive from GitHub Releases and add the extracted directory to your `PATH`.
